# Projeto Final de Programação Multiplataforma - Módulo 4.1

> Projeto desenvolvido por Ana Julia Lima de Oliveira como um Produtor Kafka.

> Este módulo tem outros dois Consumidores Kafka (4.2 e 4.3)

---

## Como iniciar o projeto localmente

### 1. Git clone
```bash
git clone https://github.com/AnaJulia-FATEC-Projeto-Final/modulo4.1.git

cd modulo4.1
````

### 2. Construir containers e Rodar via IDE (recomendado)
```bash
docker-compose up -d

# Abra a IDE (IntelliJ recomendado) e importe o projeto
```

### 3. Rode os dois Consumers e veja os logs
